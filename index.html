Introduction à Firebase

Firebase est une plateforme de développement d'applications web et mobiles développée par Google. Elle offre un ensemble complet de services backend pour accélérer votre développement.

Avantage clé : Concentrez-vous sur l'expérience utilisateur pendant que Firebase gère l'infrastructure complexe.

Histoire et Évolution

2011

Création de Firebase

Lancé comme une base de données temps réel

2014

Acquisition par Google

Intégration à la suite Google Cloud

2023

Plateforme mature

Plus de 20 services intégrés

Fonctionnalités Clés

Firestore

Base de données NoSQL flexible et scalable avec synchronisation en temps réel

Système d'authentification prêt à l'emploi avec multiples fournisseurs

Stockage sécurisé pour les fichiers utilisateurs avec règles granulaires

Suivi complet du comportement des utilisateurs

Premiers Pas avec Firebase

Initialisation
Authentification
Firestore
// Installer Firebase
npm install firebase

// Initialiser Firebase
import { initializeApp } from "firebase/app";

const firebaseConfig = {
  apiKey: "VOTRE_CLE_API",
  authDomain: "VOTRE_DOMAINE.firebaseapp.com",
  projectId: "VOTRE_PROJECT_ID",
  storageBucket: "VOTRE_BUCKET.appspot.com",
  messagingSenderId: "VOTRE_SENDER_ID",
  appId: "VOTRE_APP_ID"
};

const app = initializeApp(firebaseConfig);
// Configuration Auth
import { getAuth, signInWithEmailAndPassword } from "firebase/auth";

const auth = getAuth();
signInWithEmailAndPassword(auth, email, password)
  .then((userCredential) => {
    // Utilisateur connecté
  })
  .catch((error) => {
    // Gestion des erreurs
  });
// Utilisation Firestore
import { getFirestore, doc, setDoc } from "firebase/firestore";

const db = getFirestore(app);

// Ajouter un document
await setDoc(doc(db, "users", "user123"), {
  name: "John Doe",
  email: "john@example.com",
  createdAt: new Date()
});
